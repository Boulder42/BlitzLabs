Welcome!

Blitz Labs is a real-time statistic-tracking website started by BearHawkMan in early 2015. This site was started as an improvement over wotbstars.com, which had a very basic interface and was not very appealing. At the time I had very little programming knowledge, but I grew and cultivated that so I can build the website you know and love today! I currently run this website as sort of a "side hustle", it is not my full time job, so if you would like to support the website, you can donate here.
